# Trellis Sequencer

This project uses NPM Workspaces and Concurrently to run the front and backend simultaneously

## Tech Stack

### Core Logic & UI
- React
- Tone.js

### Synchronization
- Socket.io
- Yjs

### Infrastructure
- Server-side logic:
  - Node.js & Express
- Database / storage
  - MongoDB / PostgresSQL

- Concurrently


## Run the application
- Navigate to the root directory
  bash
  npm run dev
